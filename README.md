# CyberTank
autonomous multi role ground vehicle

## Components required 
1. Huskylens AI camera
2. Arduino Mega
3. L298N motor driver
4. dc motor * 2
5. Keypad 4*3
6. Ultrasonic (yet to give)
7. LiPo 2200Mah
8. power supply for Arduino ( Power Bank)
